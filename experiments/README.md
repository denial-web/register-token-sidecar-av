# Experiments

This folder is reserved for offline validation work.

Suggested first steps:

1. Add a baseline trigger script or notebook.
2. Add one small public dataset workflow, such as `nuScenes mini` or `BDD100K`.
3. Record evaluation settings so results are reproducible.

Suggested contents:

- `baselines/` for heuristic trigger implementations
- `notebooks/` for quick analysis
- `configs/` for repeatable experiment settings
- `metrics.md` for metric definitions and reporting rules
